Instructions to run the application
1. Clone the repository using the command git clone https://github.com/tswinyane/users-app.git
2. Inside users-app directory take the usersdb.sql file which is a dump of database, it contains a table called user with 3 records and import it into mysql database

3. Change the connection settings inside the conn file to your own login ones
4. If you are using linux server, you can put the users-app directory inside the /var/www/html/ folder
5. The on your browser navigate to http://localhost/users-app to run the application
